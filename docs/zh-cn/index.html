🌟 强大功能

🎬

Sora AI 视频生成

使用 OpenAI 尖端的 Sora 模型将文本描述转换为令人惊叹的视频。创建具有可定制持续时间和纵横比的高质量视频。

  • 文本转视频生成
  • AI 提示增强
  • 进度跟踪
  • 高质量输出
🎙️

GPT-4o 实时语音聊天

使用 GPT-4o-realtime-preview 模型体验实时语音交互的自然对话。自然交谈并获得即时回应。

  • 实时语音对话
  • 自然语音识别
  • 即时 AI 回应
  • 移动端优化
🧠

最新 AI 模型

支持最先进的 AI 模型，包括 GPT-4o、GPT-4o-mini、o1、o1-mini、o3、o3-mini 等。

📱

渐进式网络应用

可在任何设备上作为原生应用安装。针对移动设备和平板电脑优化，具有离线功能。

🖼️

图像生成与编辑

使用 DALL-E 和 GPT-Image-1 API 生成和编辑图像。从文本描述创建视觉内容。

🗣️

语音服务

Azure 文本转语音和语音转文本集成，支持使用 Whisper 模型的批量转录。

🔐

Azure AD 身份验证

使用 Azure Active Directory 进行安全身份验证，具有企业级安全功能。

💬

高级对话控制

完全控制消息，具有删除、隐藏/取消隐藏功能。将对话导出为 markdown。

📰

Bing 新闻集成

通过 Bing API 集成直接在对话中获取最新新闻更新。

🚀 快速开始

1

克隆仓库

从 GitHub 获取源代码

git clone https://github.com/hddevteam/Azure-chatGPT-demo.git
2

配置环境

设置您的 Azure OpenAI API 密钥和端点

cp .envtemplate .env # 使用您的 Azure OpenAI 凭据编辑 .env
3

安装依赖

安装所需的 Node.js 包

npm install
4

启动应用程序

启动开发服务器

npm start

前提条件

  • 安装 Node.js 16+
  • Azure OpenAI API 访问权限
  • Azure 订阅（用于高级功能）
  • Sora API 访问权限（用于视频生成）
